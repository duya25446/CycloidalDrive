# 摆线减速器脚本使用教程

本文档介绍 Fusion 360 中的摆线减速器生成脚本的使用方式与参数含义。

## 1. 前置条件
- Fusion 360 已打开一个设计文档（零件或装配均可）
- 脚本已加载并可执行

## 2. 运行脚本
1. 打开脚本面板，运行“创建摆线减速器”
2. 出现参数对话框后，按需要填写参数
3. 勾选“实时预览”可边改边更新（仅用于快速查看）
4. 点击确定，生成草图与组件内容

## 3. 必要参数
- **减速比**：减速比 = 外针数量 - 1，等同于摆线轮齿数
- **偏心量**：偏心距离
- **外针直径**：外针的直径
- **外针节圆直径**：外针中心所在节圆直径
- **每齿采样点数**：每个齿的曲线采样点数，越大越平滑
- **最小压力角**：只读显示，脚本自动计算

## 4. 可选参数
### 4.1 摆线轮中心孔
- **绘制中心孔**：是否绘制中心孔
- **直径**：中心孔直径

### 4.2 摆线轮与输出盘
- **绘制周孔**：是否绘制摆线轮周孔
- **绘制输出盘销**：是否绘制输出盘销
- **设置对象**：切换“摆线轮孔 / 输出盘销”以编辑对应参数

摆线轮孔参数：
- **孔数量**：孔数量
- **孔直径**：孔直径
- **中心到孔距**：孔中心与轴心距离

输出盘销参数：
- **销数量**：销数量
- **销直径**：销直径
- **中心到销距**：销中心与轴心距离

## 5. 常见问题
- **无法新建组件**：在零件设计中会直接使用根组件绘制
- **参数报错**：确认所有直径与距离均大于 0
